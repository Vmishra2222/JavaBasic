
# Java Spring Boot REST API Coursework

This repository contains the coursework and practical exercises from a comprehensive Java Technical Training. The primary focus of these projects is to understand the fundamentals of building RESTful web services and managing project configurations.

## 🚀 Key Concepts Learned

* **Dependency Management:** Configuring and managing project dependencies using the `pom.xml` file, understanding Maven lifecycles, and integrating Spring Boot starter packs.
* **REST Controllers:** Utilizing the `@RestController` annotation to create scalable and stateless web services that handle HTTP requests and return JSON responses.
* **Request & URL Mapping:** * Implementing `@RequestMapping` for class-level routing.
    * Handling specific HTTP methods using annotations like `@GetMapping`, `@PostMapping`, `@PutMapping`, and `@DeleteMapping`.
    * Managing dynamic URLs with `@PathVariable` and `@RequestParam`.

## 🛠️ Tech Stack

* **Language:** Java
* **Framework:** Spring Boot
* **Build Tool:** Maven
